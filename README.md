## capture.py
左右カメラにて画像を撮像する．

## stereo_rectify.py
左右画像の平行化を行う．

## block_match.py
SADにてテンプレートマッチングを行う．

## ran.py
RANSACアルゴリズムにてノイズ除去を行う．

## plot.py
三次元座標を算出し，CloudCompareにて三次元復元を行うためのptsファイルを生成する．
